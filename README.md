# Sveltekit-Tailwind-Starter

My personal sveltekit template.

## To clone this repo

Run

```
git clone https://github.com/sammi-turner/Sveltekit-Tailwind-Starter/
```

## To install node modules

From the root directory, run

```
pnpm i
```


## To run the dev server

From the root directory, run

```
pnpm run dev
```
